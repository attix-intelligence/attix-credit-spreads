## 1. Top-Line Findings

1. **`pytest.ini` does not measure `strategies/`** — the directory containing 10 strategy classes (2,494 LOC) is missing from `--cov=` flags. Coverage there averages **27%** when measured, but in CI it's invisible.
2. **`PRODUCTION_MANIFEST.md` (2026-04-23) is stale** — 10 of the 42 modules it lists as production have since been moved to `compass/archive/`. The newer `docs/DEAD_CODE_INVENTORY.md` (2026-04-24) is closer to truth but is also drifting.
3. **`main.py` has hard import dependencies on archived modules** — SLOT_RETRAIN code path will crash at runtime (`compass.retrain_scheduler` exists only in `compass/archive/`).
4. **Critical execution paths are weakly covered** — `alpaca_connector.py` (437 LOC) at **0%**, `shared/iron_vault.py` (the no-synthetic-pricing source) at **35%**, `compass/portfolio_risk_manager.py` (208 LOC) at **0%**.
5. **2 failing tests on `master`** — not skipped, not xfailed, just failing.
