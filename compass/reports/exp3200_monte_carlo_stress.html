⚠ Modelling caveats. (1) MVN — no t-tail or jump-diffusion; real options-strategy returns are heavier-tailed, so this is an optimistic simulator. (2) Flash-crash uses linear β-propagation of the SPY shock; nonlinear correlation-breakdown during the shock is not captured (covered separately by credit_freeze). (3) Calibration window is post-2020 only — pre-2020 IronVault data is not in this repo, so we cannot test pre-2010 regimes.
